# Zenith-Analysis

[English](./README.md) | [简体中文](./README.zh-CN.md) | [日本語](./README.ja.md)

Zenith-Analysis 是一个可自行部署的股票技术分析工作台，覆盖美股、港股、中国 A 股和日本股票市场。它在一个响应式仪表盘中整合多周期行情、基于规则的技术证据、买点魅力评分、交互式图表，以及可选的大语言模型研报。

本项目仅用于研究和学习，不构成投资建议，也不保证第三方行情数据的准确性或可用性。

## 主要功能

- 搜索支持市场中的股票，并在浏览器中保存分析历史和行情快照。
- 在行情源可用时，基于真实市场数据构建日线和周线技术分析快照。
- 展示联动的价格与指标图层，支持日线/周线切换，以及红涨或绿涨配色。
- 根据明确的规则证据和数据质量约束，计算 0-5 分的买点魅力评分。
- 允许已配置的大语言模型复核证据，对规则评分进行最多 `+/-0.5` 的调整。
- 分别生成行情概览、策略建议和技术细节三部分研报。
- 大模型生成失败且已开启兜底时，自动使用内置本地研报引擎。
- 实时行情源无法返回有效结果时，明确标记离线模拟数据。

## 分析范围

| 类别 | 分析内容 |
| --- | --- |
| 趋势与波动 | EMA、布林带、Ichimoku 一目均衡表、ATR、日线/周线结构 |
| 动量 | MACD、KDJ、RSI、近期交叉与背离 |
| 成交量与资金流 | 成交量均线、OBV、CMF、量价确认 |
| 价格位置 | 水平支撑/压力、斐波那契回撤、VPVR 价值区和主要筹码节点 |
| 结构与形态 | 经典图表形态、结合上下文的 K 线形态、TD Sequential 神奇九转 |
| 市场结构 | 艾略特波浪启发式分析，以及缠论分型、笔和中枢背景 |
| 决策辅助 | 入场情景、失效位、目标位、风险收益背景、数据质量评分上限 |

经典结构识别包括双顶/双底、三重顶/三重底、头肩形态、杯柄、圆弧结构、旗形、矩形、三角形、三角旗形和上升/下降楔形。形态结果只是分析证据，不能单独视为交易信号。

## 市场与行情数据

| 市场 | 代码示例 |
| --- | --- |
| 美股 | `AAPL`、`MSFT` |
| 港股 | `0700.HK`、`9988.HK` |
| 中国 A 股 | `600519.SS`、`000001.SZ` |
| 日本股票 | `7203.T`、`9984.T` |

行情可用性取决于股票、市场、网络和上游服务。应用会根据市场和接口，在 Yahoo Finance、东方财富、同花顺、Kabutan、腾讯，以及可选的 Twelve Data/FMP 之间执行降级。不同接口和市场的调用顺序可能不同。

在支持的市场中，实时行情会合并到分析快照，使当前价格、涨跌幅、指标和评分使用一致的最新 K 线。最后级别的兜底数据或模拟数据不会作为主要行情长期缓存。

## AI 研报与评分

分析引擎会先构建不可变的技术证据快照，再计算确定性的买点魅力评分。评分会考虑趋势、动量、支撑压力、波动率、成交量、VPVR、斐波那契位置、形态背景和数据质量。

配置大语言模型后，模型收到的是结构化证据，而不是被要求自行编造或重新计算指标。模型只能在 `+/-0.5` 范围内复核规则评分，并被要求引用已有证据。最终研报分为：

- 市场与交易形态概览；
- 策略、触发条件和风险控制；
- 详细技术证据。

界面可选择 Google Gemini、OpenAI 和 Anthropic，也可以为 OpenAI 兼容服务填写自定义 Base URL。如果未配置大模型，或大模型请求失败且已启用本地兜底，仍可使用内置研报引擎。

## 快速开始

### 环境要求

- Node.js 20.9 或更高版本
- npm

### 安装与运行

```bash
git clone https://github.com/gunzero5158/zenith-quant.git
cd zenith-quant
npm install
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000)。

生产环境构建：

```bash
npm run build
npm run start
```

## 配置

### 大语言模型

在应用中打开 **AI 设置**，填写：

- 服务提供商；
- 模型名称；
- API Key；
- 可选的 API Base URL；
- 是否启用本地研报兜底。

浏览器会在本地保存这些配置。执行分析时，凭据会发送到本应用的服务端接口，再转发至所选的大模型上游服务。本应用不会主动在服务端持久化保存大模型凭据。

默认禁止访问私有或内部的大模型地址。确实需要连接内网模型的自行部署环境可以显式开启：

```env
ZENITH_ALLOW_PRIVATE_LLM_HOSTS=true
```

### 可选行情源

创建 `.env.local` 可启用额外的兜底行情源：

```env
TWELVE_DATA_API_KEY=your_key
FMP_API_KEY=your_key
```

这两个密钥均为可选项。不配置时，应用仍会使用内置行情源链，但实际结果取决于上游服务是否可用。

## 隐私与限制

- 分析历史、缓存研报、显示偏好和大模型设置保存在浏览器存储或 Cookie 中。
- 行情和 AI 输出来自外部服务，可能存在延迟、缺失、不可用或错误。
- 出现演示模式提示时，当前结果使用的是模拟数据，而非实时行情。
- 本地缓存可以减少重复请求，也可以通过强制刷新发起新的分析。
- 技术指标、形态、评分和生成式研报只是概率性的决策辅助工具，不是价格预测或交易指令。
