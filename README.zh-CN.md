# Zenith-Quant (天顶量化) 📈

[English](./README.md) | [日本語](./README.ja.md)

Zenith-Quant 是一款专为股票投资者打造的**多维度、多指标、多形态技术面深度综合分析系统**。系统融合了经典的量化技术指标、高级趋势理论（艾略特波浪理论 & 缠论）、支撑压力筹码分布以及大语言模型（LLM）的智能研判，为您提供极具金融质感和专业度的股票深度技术研报。

---

## 🌟 核心卖点

### 1. 多维度算法研判矩阵
我们不依赖单一的指标，而是通过以下五大维度进行全面剖析：
* **基础技术指标**：EMA 均线系统、Bollinger Bands (布林线)、MACD 震荡指标、KDJ 随机指标、RSI 强弱指标、ATR 真实波幅。
* **量能力度与资金流**：结合 OBV (能量潮) 与 CMF (蔡金资金流量)，辅以成交量 SMA 均线，识别主力资金异动与量价背离。
* **经典趋势与形态学**：
  * **神奇九转 (TD Sequential)**：精准捕捉市场加速与拐点。
  * **经典形态识别**：双底 (W底)、头肩顶/底、杯柄形态、圆弧顶/底等。
  * **斐波那契回撤**：基于历史波段自动绘制黄金分割支撑/阻力线。
* **波浪理论 (Elliott Wave Theory)**：自动匹配历史极值，识别当前的脉冲主升浪 (Wave 3/5) 或 ABC 调整浪，研判所处位置。
* **缠论解析 (Chanlun)**：内置经典的缠论算法，自动处理 K 线包含关系，扫描顶底分型 (Ding/Di FenXing) 并连线画笔 (Strokes)。
* **筹码分布与支撑压力**：自动聚类历史高低极值，计算 POC (控制筹码中心点) 以及动态/静态支撑与阻力位。

### 2. AI 深度技术研报
* 汇聚上述数十项客观指标，通过底层评分矩阵进行量化打分。
* 将深度结构化的指标数据作为 Payload 注入大模型 API，自动生成集“综合概述”、“交易策略建议”和“技术解构”于一体的专业深度研报。
* 支持多语言（英文、简体中文、繁体中文、日文）输出。

### 3. 智能本地缓存与防冗余请求 (Local Caching)
* **股市非交易时段智能识别**：系统内置对 **A股、港股、美股、日股** 的交易时段判定（包括双休、盘中休市与盘后收盘时间，并自动转换为北京时间）。
* **0.1s 秒开无感体验**：点击分析历史时，若检测到该股同日内已有缓存且当前处于对应股市的非交易时段，则跳过网络请求直接从本地缓存恢复 K 线图与 AI 研报，省时且节约 API Token 开销。
* **强制刷新**：保留手动刷新按钮，允许用户一键穿透缓存获取最新瞬时数据。

### 4. 极致视觉体验与细节控制
* **首创红绿涨跌反转器**：支持一键切换“红涨绿跌”或“绿涨红跌”配色，图表、个股现价涨跌幅、分析历史列表配色一键全局同步联动。
* **极客科技风 UI**：基于玻璃拟态 (Glassmorphic) 质感与深色调，配以发光 SVG 趋势 Logo、平滑的加载过渡动画以及模拟黑客帝国的分析终端运行日志输出。

## 🚀 快速开始

### 运行环境
* Node.js >= 18.0.0
* 推荐使用 NPM / PNPM 包管理工具

### 安装与启动
1. 克隆或下载本项目至本地目录。
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动开发服务器：
   ```bash
   npm run dev
   ```
4. 运行单元测试：
   ```bash
   npm run test:run
   ```
5. 在浏览器中打开 [http://localhost:3000](http://localhost:3000) 即可开始使用。
