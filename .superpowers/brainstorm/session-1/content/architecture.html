选择项目架构 / Framework Selection

请选择您期望的股票分析网站的技术架构体系

A

Next.js (App Router) + Vanilla CSS (推荐)

统一架构：前端 React 与后端 API 路由在同一个项目。使用服务器端渲染（SSR）或 API 路由直接调用 yahoo-finance2，完美规避前端 CORS 跨域限制，对 API 密钥（若后续使用）保护极佳。

B

Vite (React) + Express (Node) + Vanilla CSS

前后端分离：前端是纯粹的单页应用（SPA），使用 Vite 构建；后端使用 Express 提供 API 服务。适合喜欢传统前后端完全独立部署和开发模式的场景。

API 选型建议：

我们将使用完全免费、无需 API 密匙、支持全球市场（美股、港股、A股、日股）的 yahoo-finance2 Node.js 库。无论是 Next.js 的 API 路由还是 Express 都可以轻松集成。
