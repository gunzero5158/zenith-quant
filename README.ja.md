Zenith-Analysis (天頂分析) は、個人およびプロの投資家向けに構築された、**多次元・多指標・多パターンの総合的テクニカル分析システム**です。伝統的なテクニカル指標、高度なトレンド理論（エリオット波動理論 ＆ チャンルン（纏論））、価格帯別出来高の支持・抵抗帯解析、そして大画面言語モデル（LLM）による知的な市況判断を組み合わせ、極めてプロフェッショナルで洗練されたディープ・テクニカル分析レポートを瞬時に生成します。

---

## 🌟 主な機能と特徴

### 1. 多次元アルゴリズム判定マトリクス
単一の指標に依存せず、以下の5つの次元から多角的に株価を徹底解剖します：
* **基礎テクニカル指標**：EMA移動平均線システム、ボリンジャーバンド（Bollinger Bands）、MACDオシレーター、KDJストキャスティクス、RSI相対力指数、ATR（Average True Range）。
* **出来高分析と資金フロー**：OBV（オン・バランス・ボリューム）とCMF（チャイキン・マネー・フロー）に20日出来高移動平均（SMA）を組み合わせ、クジラ（大口投資家）の資金流入や価格と出来高の逆行（ダイバージェンス）を検出します。
* **トレンドとパターン認識**：
  * **TDシーケンシャル（神奇九転）**：相場の加速とトレンド転換点を高精度で検出。
  * **チャートパターン認識**：ダブルボトム（W底）、三尊天井（ヘッド＆ショルダー）、カップウィズハンドル、ソーサートップなどの伝統的パターンを自動識別。
  * **フィボナッチ・リトレースメント**：過去の変動レンジに基づき、フィボナッチ黄金比率による支持線・抵抗線を自動描画。
* **エリオット波動理論 (Elliott Wave Theory)**：過去のスイングハイ・ローをマッチングし、現在が上昇推進波（第3波/第5波）か、調整波（ABC波）かを自動分類します。
* **チャンルン（纏論）解析**：チャンルンアルゴリズムを実装し、ローソク足の「包含関係」を自動処理した上で、頂・底分型（Ding/Di FenXing）を検出し、筆（Strokes）を描画します。
* **価格帯別出来高（POC）と支持・抵抗帯**：過去の価格分布と出来高から最も取引が集中した価格帯（POC：Point of Control）および動的・静的な支持・抵抗帯を自動計算します。

### 2. AIによるディープ・テクニカル分析レポート
* 上記の数十種類の客観的データを内部スコアリングシステムで数値化。
* 構造化された指標データをペイロードとして大口LLMのAPIに投入し、「総合評価」、「推奨取引戦略」、「テクニカル構造の解体」を盛り込んだ金融機関品質の専門レポートを生成。
* 多言語（英語、簡体字中国語、繁体字中国語、日本語）出力に対応。

### 3. インテリジェント・ローカルキャッシュと冗長リクエスト防止 (Local Caching)
* **各国市場の取引時間判定**：**日本株（TSE）、中国A株、香港株、米国株** の取引時間、昼休み、取引終了時間（UTC+8北京時間に自動変換）をインテリジェントに判定。
* **0.1秒爆速起動**：過去に分析した銘柄を再選択した際、同日内のデータであり、かつ該当市場の取引時間外である場合は、APIリクエストをスキップして即座にローカルキャッシュからチャートとAIレポートを復元。APIトークン費用を節約します。
* **強制リロード**：キャッシュをバイパスして最新のリアルタイムデータを取得するための「強制更新」ボタンを完備。

### 4. プレミアムなビジュアル体験
* **チャートカラー反転機能**：ワンクリックで「赤上昇・緑下落（日本・中国仕様）」と「緑上昇・赤下落（米国・グローバル仕様）」の配色を切り替えます。チャートだけでなく、現在の株価の騰落率や履歴リストの配色まで一瞬で完全連動します。
* **洗練されたダークUI**：グラスモフィズム（Glassmorphic）を取り入れたダークスタイル、光るSVGトレンドロゴ、滑らかな読み込みアニメーション、そしてハッカー端末風にアルゴリズムの稼働ログが流れるターミナル出力を搭載。

## 🚀 クイックスタート

### 動作環境
* Node.js >= 18.0.0
* パッケージ管理ツールとして NPM または PNPM を推奨

### インストールと起動
1. 本プロジェクトをローカルディレクトリにクローンまたはダウンロードします。
2. 依存関係のインストール：
   ```bash
   npm install
   ```
3. 開発サーバーの起動：
   ```bash
   npm run dev
   ```
4. テストの実行：
   ```bash
   npm run test:run
   ```
5. ブラウザで [http://localhost:3000](http://localhost:3000) を開くとシステムを使用できます。
